# SistemaInventarioGit

## Descripción
Pequeño sistema en lenguaje C para registrar equipos informáticos de una empresa de soporte técnico, al mismo tiempo el proyecto que debemos realizar el dia de hoy.

## Funcionalidades
- Registrar equipo
- Mostrar equipo registrado
- Modificar equipo
- Eliminar equipo

## Tecnologías utilizadas
- Lenguaje C
- Git
- GitHub
